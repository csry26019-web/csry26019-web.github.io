# csry26019-web.github.io

幕田 亮介（Ryosuke Makuta）の研究ポートフォリオサイトです。
日本大学大学院 理工学研究科 修士課程（高橋研究室、指導教員：高橋 聖 教授）。

深層強化学習・階層型強化学習を用いた交通信号制御に関する研究を紹介しています。

## 公開URL

https://csry26019-web.github.io/

> このサイトは検索エンジンにインデックスされない設定（`noindex`）にしており、
> URL を知っている人だけが閲覧する想定で公開しています。

## 構成

- `index.html` … サイト本体（HTML / CSS / JavaScript を1ファイルにまとめています）
- `.nojekyll` … GitHub Pages の Jekyll 処理を無効化するためのファイル

## 更新方法

1. `index.html` を編集して保存する
2. ブラウザで表示を確認する
3. 変更をコミットしてプッシュする（`git add` → `git commit` → `git push`）
4. 数分後に公開URLで反映を確認する
